# Python & Data Science - Projects

Hands-on projects from Python fundamentals to deep learning.

## Project Areas
1. **Python Labs** - Language fundamentals, scripting, OOP exercises
2. **Data Analysis Projects** - EDA, cleaning, visualization on real datasets
3. **ML Projects** - End-to-end ML pipelines with real-world data
4. **Deep Learning Projects** - Neural networks, NLP, computer vision

## Setup
```bash
python -m venv venv
source venv/bin/activate  # macOS/Linux
pip install -r requirements.txt
```

## Recommended Order
1. Start with Python Labs (language mastery)
2. Move to Data Analysis (NumPy, Pandas, visualization)
3. Build ML projects (Scikit-Learn pipelines)
4. Advance to Deep Learning (TensorFlow/PyTorch)
