# Data Analysis Projects - Curriculum

Real-world data analysis projects using NumPy, Pandas, and visualization libraries.

---

## Project 01: NumPy Fundamentals
- [ ] **Image Processor**: load images as NumPy arrays, apply grayscale, blur, crop, rotate
- [ ] **Linear Regression from Scratch**: implement using NumPy (no Scikit-Learn)
- [ ] **Random Walk Simulator**: visualize 1D and 2D random walks
- [ ] **Statistical Calculator**: mean, median, mode, std, correlation on a dataset

## Project 02: Pandas EDA (Exploratory Data Analysis)
- [ ] **Titanic Dataset EDA**: survival analysis by class, gender, age
- [ ] **COVID-19 Dashboard Data**: time series analysis of cases, deaths, vaccinations
- [ ] **E-Commerce Analysis**: RFM analysis (Recency, Frequency, Monetary)
- [ ] **Stack Overflow Survey**: developer trends, salary analysis, technology popularity

## Project 03: Data Cleaning Pipeline
- [ ] **Messy Sales Data**: fix missing values, duplicates, inconsistent formats
- [ ] **Web Scraped Data**: clean HTML artifacts, standardize columns, handle encoding
- [ ] **Multi-Source Merge**: combine data from CSV, JSON, and Excel into unified dataset
- [ ] **Automated Cleaning Pipeline**: reusable `DataCleaner` class with method chaining

## Project 04: Advanced Pandas
- [ ] **Stock Market Analysis**: moving averages, daily returns, volatility, correlation heatmap
- [ ] **Retail Sales Forecasting**: groupby, pivot tables, seasonal patterns
- [ ] **A/B Test Analysis**: statistical significance, p-values, confidence intervals
- [ ] **Geospatial Analysis**: merge datasets with location data, visualize on map

## Project 05: Visualization Portfolio
- [ ] **Matplotlib**: publication-quality multi-panel figure (4+ subplots)
- [ ] **Seaborn**: complete EDA report with distribution, correlation, categorical plots
- [ ] **Plotly Interactive**: interactive dashboard with dropdowns, sliders, hover
- [ ] **Streamlit App**: deploy an interactive data exploration app
- [ ] **Storytelling Report**: Jupyter notebook with narrative, insights, and recommendations

---

- Matplotlib, Seaborn, Plotly
