# Structural Design Patterns - Curriculum

## Topics
- [ ] **Adapter**: convert interface to expected interface
- [ ] **Bridge**: decouple abstraction from implementation
- [ ] **Composite**: tree structures, uniform treatment of leaf and composite
- [ ] **Decorator**: add behavior dynamically (wrapping)
- [ ] **Facade**: simplified interface to complex subsystem
- [ ] **Flyweight**: share common state to reduce memory
- [ ] **Proxy**: control access to an object
- [ ] Java examples: `InputStream` decorators, `Collections.unmodifiableList()` proxy
- [ ] Spring Boot: `@Transactional` proxy, `RestTemplate` adapter
