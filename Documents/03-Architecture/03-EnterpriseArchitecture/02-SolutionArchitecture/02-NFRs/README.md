# Non-Functional Requirements (NFRs) - Curriculum

## Topics
- [ ] Performance: latency, throughput, response time targets
- [ ] Scalability: horizontal vs vertical, auto-scaling triggers

- [ ] Security: authentication, authorization, encryption, compliance
- [ ] Reliability: fault tolerance, disaster recovery
- [ ] Maintainability: code quality, deployment frequency, MTTR
- [ ] Observability: logging, metrics, tracing standards
- [ ] Testability: test coverage, test automation requirements
- [ ] Quantifying NFRs: measurable, testable criteria
- [ ] NFR trade-offs and prioritization
