# Microservice Patterns

Patterns for building resilient, scalable microservices. 23 patterns, ~107 modules.

## Sections
1. **Core Patterns** (11 topics) - API Gateway, Service Mesh, Sidecar, BFF, Saga, Database per Service, Service Decomposition, Leader Election, Service Discovery, Health Check, API Composition
2. **Resilience** (7 topics) - Circuit Breaker, Retry & Backoff, Bulkhead, Rate Limiting, Idempotency, Distributed Lock, Backpressure
3. **Data Patterns** (5 topics) - CQRS, Event Sourcing, Outbox, Strangler Fig, Event-Driven Patterns
