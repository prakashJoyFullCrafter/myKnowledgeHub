# Data Visualization - Curriculum

## Module 1: Matplotlib
- [ ] Figure and Axes: `plt.figure()`, `plt.subplots()`
- [ ] Line plots, bar charts, scatter plots, histograms
- [ ] Customization: titles, labels, legends, colors, styles
- [ ] Multiple subplots: `fig, axes = plt.subplots(2, 2)`
- [ ] Saving figures: `savefig()`
- [ ] `plt.style.use()` for themes

## Module 2: Seaborn
- [ ] Built on Matplotlib, statistical visualization
- [ ] Distribution plots: `histplot`, `kdeplot`, `boxplot`, `violinplot`
- [ ] Categorical plots: `countplot`, `barplot`, `stripplot`, `swarmplot`
- [ ] Relationship plots: `scatterplot`, `lineplot`, `regplot`, `lmplot`
- [ ] Matrix plots: `heatmap`, `clustermap`
- [ ] `pairplot` and `jointplot` for multi-variable exploration
- [ ] FacetGrid for multi-panel plots
- [ ] Color palettes and themes

## Module 3: Advanced Visualization
- [ ] Plotly for interactive plots
- [ ] Plotly Express: quick interactive charts
- [ ] Dashboards with Plotly Dash or Streamlit
- [ ] Choosing the right chart for your data
- [ ] Visualization best practices: declutter, color wisely, tell a story
- [ ] Jupyter notebook inline plotting
