# NumPy - Curriculum

## Module 1: Array Basics
- [ ] `np.array()`, `np.zeros()`, `np.ones()`, `np.arange()`, `np.linspace()`
- [ ] Array attributes: `shape`, `dtype`, `ndim`, `size`
- [ ] Reshaping: `reshape()`, `flatten()`, `ravel()`
- [ ] Indexing and slicing (1D, 2D, 3D)
- [ ] Boolean indexing and fancy indexing
- [ ] Copy vs view (`copy()` vs slicing)

## Module 2: Vectorized Operations
- [ ] Element-wise operations: `+`, `-`, `*`, `/`, `**`
- [ ] Broadcasting rules
- [ ] Universal functions (ufuncs): `np.sqrt`, `np.exp`, `np.log`
- [ ] Aggregation: `sum`, `mean`, `std`, `min`, `max`, `argmin`, `argmax`
- [ ] Axis parameter for row/column operations

## Module 3: Linear Algebra
- [ ] Dot product: `np.dot()`, `@` operator
- [ ] Matrix operations: `np.linalg.inv()`, `np.linalg.det()`, `np.linalg.eig()`
- [ ] Matrix multiplication vs element-wise
- [ ] Solving linear equations: `np.linalg.solve()`
- [ ] SVD and eigenvalue decomposition

## Module 4: Practical NumPy
- [ ] Random numbers: `np.random.rand`, `normal`, `randint`, `choice`, `seed`
- [ ] Sorting: `np.sort()`, `np.argsort()`
- [ ] Stacking: `np.vstack()`, `np.hstack()`, `np.concatenate()`
- [ ] `np.where()` for conditional selection
- [ ] `np.unique()` for unique values
- [ ] Structured arrays
- [ ] Performance: NumPy vs Python loops
