# Data Cleaning - Curriculum

## Topics
- [ ] Identifying missing data: `isnull()`, `notnull()`, `info()`
- [ ] Handling missing values: `dropna()`, `fillna()`, interpolation
- [ ] Imputation strategies: mean, median, mode, KNN imputation
- [ ] Detecting outliers: IQR method, Z-score, box plots
- [ ] Handling outliers: cap, remove, transform
- [ ] Duplicate detection and removal: `duplicated()`, `drop_duplicates()`
- [ ] Data type conversions: `astype()`, `pd.to_numeric()`, `pd.to_datetime()`
- [ ] String cleaning: strip whitespace, standardize case, fix typos
- [ ] Encoding categorical variables: label encoding, one-hot encoding (`get_dummies()`)
- [ ] Feature scaling: standardization (Z-score), normalization (Min-Max)
- [ ] Handling inconsistent data formats
- [ ] Data validation pipelines
- [ ] `missingno` library for visualizing missing data
