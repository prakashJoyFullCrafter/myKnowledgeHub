# Python File I/O & Exceptions - Curriculum

## Module 1: File Operations
- [ ] `open()`, `read()`, `write()`, `close()`
- [ ] `with` statement (context manager) - always use this
- [ ] Reading modes: `r`, `w`, `a`, `rb`, `wb`
- [ ] `readlines()`, `readline()`, iterating line by line
- [ ] `pathlib.Path` - modern path handling
- [ ] `os` and `shutil` for file system operations

## Module 2: Data Formats
- [ ] JSON: `json.loads()`, `json.dumps()`, `json.load()`, `json.dump()`
- [ ] CSV: `csv.reader`, `csv.writer`, `csv.DictReader`
- [ ] YAML: `pyyaml` library
- [ ] Pickle: serialization (security warning)
- [ ] Working with `.env` files (`python-dotenv`)

## Module 3: Exception Handling
- [ ] `try/except/else/finally`
- [ ] Catching specific exceptions
- [ ] Exception hierarchy (`BaseException` -> `Exception`)
- [ ] Custom exceptions
- [ ] `raise` and re-raising exceptions
- [ ] Context managers with `contextlib`
- [ ] `logging` module: levels, handlers, formatters
