# Python Core

Python language fundamentals for data science and backend development.

## Topics
1. **Syntax & Data Types** - Variables, types, collections, comprehensions
2. **Functions & Decorators** - Functions, closures, decorators, functools
3. **OOP** - Classes, inheritance, magic methods, dataclasses
4. **File I/O & Exceptions** - File handling, JSON/CSV, exception handling
5. **Generators & Iterators** - Lazy evaluation, itertools, context managers
