# Computer Vision - Curriculum

## Module 1: CNN Fundamentals
- [ ] Convolution operation: filters, stride, padding
- [ ] Pooling: max pooling, average pooling
- [ ] CNN architecture: Conv -> ReLU -> Pool -> Flatten -> Dense
- [ ] Classic architectures: LeNet, AlexNet, VGG, ResNet, Inception

## Module 2: Image Classification
- [ ] Transfer learning with pre-trained models (ResNet, EfficientNet, ViT)
- [ ] Data augmentation: flip, rotate, crop, color jitter
- [ ] `torchvision.transforms` / `tf.image`
- [ ] Fine-tuning strategies: freeze layers, gradual unfreezing
- [ ] Class activation maps (CAM) for interpretability

## Module 3: Object Detection & Segmentation
- [ ] Object detection: YOLO, SSD, Faster R-CNN
- [ ] Semantic segmentation: U-Net, DeepLab
- [ ] Instance segmentation: Mask R-CNN
- [ ] Bounding boxes, IoU, non-max suppression

## Module 4: Advanced Topics
- [ ] Vision Transformers (ViT)
- [ ] Generative models: GANs, VAE, Diffusion models
- [ ] OpenCV for image processing
- [ ] Real-time inference and model optimization
- [ ] Edge deployment: TFLite, ONNX
