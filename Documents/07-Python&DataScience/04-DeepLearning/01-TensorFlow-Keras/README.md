# TensorFlow & Keras - Curriculum

## Topics
- [ ] Tensors and operations
- [ ] Sequential API: `model = Sequential([Dense, Dense, ...])`
- [ ] Functional API: complex architectures (multi-input, multi-output)
- [ ] Layers: `Dense`, `Conv2D`, `LSTM`, `Dropout`, `BatchNormalization`
- [ ] Activation functions: ReLU, sigmoid, softmax, tanh
- [ ] Loss functions: MSE, cross-entropy, binary cross-entropy
- [ ] Optimizers: SGD, Adam, RMSprop, learning rate scheduling
- [ ] `model.compile()`, `model.fit()`, `model.evaluate()`, `model.predict()`
- [ ] Callbacks: `EarlyStopping`, `ModelCheckpoint`, `TensorBoard`
- [ ] Data pipelines: `tf.data.Dataset`
- [ ] Transfer learning with pre-trained models
- [ ] Model saving/loading: SavedModel, H5, TFLite
- [ ] TensorBoard for visualization
