# Natural Language Processing (NLP) - Curriculum

## Module 1: Text Preprocessing
- [ ] Tokenization, stemming, lemmatization
- [ ] Stop words removal
- [ ] Bag of Words (BoW) and TF-IDF
- [ ] Word embeddings: Word2Vec, GloVe, FastText
- [ ] `nltk` and `spaCy` libraries

## Module 2: Deep Learning for NLP
- [ ] RNNs and LSTMs for sequence modeling
- [ ] Attention mechanism
- [ ] Transformer architecture (encoder-decoder)
- [ ] Self-attention and multi-head attention
- [ ] Positional encoding

## Module 3: Transformers & LLMs
- [ ] BERT: bidirectional encoding, fine-tuning
- [ ] GPT family: autoregressive language models
- [ ] Hugging Face `transformers` library
- [ ] `pipeline()` for quick inference
- [ ] Fine-tuning pre-trained models on custom data
- [ ] Prompt engineering for LLMs
- [ ] RAG (Retrieval-Augmented Generation)

## Module 4: NLP Tasks
- [ ] Text classification (sentiment analysis)
- [ ] Named Entity Recognition (NER)
- [ ] Question answering
- [ ] Text summarization
- [ ] Machine translation
- [ ] Semantic similarity and embeddings search

| Module | Practice |
