# Model Evaluation - Curriculum

## Topics
- [ ] **Regression metrics**: MAE, MSE, RMSE, R-squared, Adjusted R-squared
- [ ] **Classification metrics**: accuracy, precision, recall, F1-score
- [ ] Confusion matrix: TP, TN, FP, FN
- [ ] ROC curve and AUC (Area Under Curve)
- [ ] Precision-Recall curve (for imbalanced data)
- [ ] Log loss (cross-entropy)
- [ ] Cross-validation: K-fold, stratified K-fold, leave-one-out
- [ ] Hyperparameter tuning: GridSearch, RandomSearch, Bayesian optimization (Optuna)
- [ ] Learning curves: diagnosing bias/variance
- [ ] Validation curves: effect of hyperparameters
- [ ] Nested cross-validation for unbiased evaluation
- [ ] A/B testing for model deployment
- [ ] Model interpretability: SHAP, LIME, feature importance
