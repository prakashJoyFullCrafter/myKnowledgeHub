# Unsupervised Learning - Curriculum

## Module 1: Clustering
- [ ] K-Means: algorithm, choosing K (elbow method, silhouette score)
- [ ] DBSCAN: density-based, no need to specify K
- [ ] Hierarchical clustering: agglomerative, dendrograms
- [ ] Gaussian Mixture Models (GMM)
- [ ] Cluster evaluation: silhouette score, Davies-Bouldin index

## Module 2: Dimensionality Reduction
- [ ] PCA (Principal Component Analysis): variance explained, scree plot
- [ ] t-SNE: visualization of high-dimensional data
- [ ] UMAP: faster alternative to t-SNE
- [ ] Feature selection vs feature extraction
- [ ] When to reduce dimensions

## Module 3: Other Unsupervised Techniques
- [ ] Anomaly detection: Isolation Forest, One-Class SVM
- [ ] Association rules: Apriori algorithm
- [ ] Topic modeling: LDA (Latent Dirichlet Allocation)
