# RabbitMQ Routing Keys - Curriculum

## Topics
- [ ] Routing key: message label for routing decisions
- [ ] Direct exchange: exact routing key match
- [ ] Topic exchange: pattern matching with `*` (one word) and `#` (zero or more)
- [ ] Examples: `order.created`, `order.*.shipped`, `audit.#`
- [ ] Headers exchange: routing by message headers (not routing key)
- [ ] Routing key design best practices
