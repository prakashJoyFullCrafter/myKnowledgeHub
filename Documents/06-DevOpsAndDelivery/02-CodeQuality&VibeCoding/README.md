# Code Quality & Vibe Coding - Expert Curriculum

Testing, quality practices, and AI-assisted development for building production-grade software.

## Sections

### 01-Testing (31 modules)
Write comprehensive tests at every level of the testing pyramid.
- **Unit Tests** - JUnit 5, Mockito, AssertJ, parameterized, coverage, mutation testing
- **Integration Tests** - Spring Boot slices, MockMvc, WireMock, database tests
- **Contract Tests** - Spring Cloud Contract, Pact, Pact Broker, CI gates
- **Testcontainers** - Real Docker dependencies, singleton pattern, Spring Boot 3.1+

### 02-Quality Practices (36 modules)
Master the craft of writing clean, maintainable, well-designed code.
- **Clean Code** - Naming, functions, error handling, code smells, clean architecture
- **TDD** - Red-Green-Refactor, Chicago vs London school, kata practice
- **BDD** - Gherkin, Cucumber, Three Amigos, living documentation
- **Code Review** - Checklist, giving/receiving feedback, automation
- **Refactoring** - Martin Fowler's catalog, legacy code, IDE tooling

### 03-AI-Assisted Coding (15 modules)
Multiply your productivity with AI tools while maintaining quality.
- **Prompt Engineering** - Code generation, debugging, architecture, prompt patterns
- **AI Code Review** - Security/performance review, CI/CD integration, custom rules
