# Data & Persistence

Database fundamentals, JDBC, PostgreSQL mastery, and Redis caching.

## Topics
1. **SQL** - Core SQL, joins, CTEs, window functions
2. **JDBC** - Connection management, PreparedStatement, transactions, batch, HikariCP, JdbcTemplate (9 modules)
3. **PostgreSQL** - Fundamentals, indexes, transactions, migrations, pooling
4. **Redis** - Data structures, caching, pub/sub, sessions, Lua scripting
