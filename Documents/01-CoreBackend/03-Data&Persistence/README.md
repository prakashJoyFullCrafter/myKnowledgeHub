3. **PostgreSQL** - PG types, JSONB, indexes, EXPLAIN, transactions, MVCC, migrations, pooling (5 topics)
4. **Redis** - Data structures, caching, pub/sub, sessions, Lua, persistence, cluster/sentinel, Spring Data Redis (8 topics)
