# PostgreSQL SQL Fundamentals - Curriculum

## Topics
- [ ] PostgreSQL-specific data types: `SERIAL`, `UUID`, `JSONB`, `ARRAY`, `INET`, `TSRANGE`
- [ ] JSONB operators: `->`, `->>`, `@>`, `?`, `jsonb_agg`
- [ ] Array operations: `ANY()`, `ALL()`, `array_agg()`
- [ ] Full-text search: `tsvector`, `tsquery`, `to_tsvector()`, `@@`
- [ ] Regular expressions: `~`, `~*`, `SIMILAR TO`
- [ ] `RETURNING` clause for INSERT/UPDATE/DELETE
- [ ] `UPSERT` with `ON CONFLICT DO UPDATE`
- [ ] Lateral joins (`LATERAL`)
- [ ] `COPY` for bulk data import/export
