# Redis

In-memory data store for caching, sessions, and messaging.

## Topics
1. **Data Structures** - Strings, lists, sets, sorted sets, hashes, streams
2. **Caching Patterns** - Cache-aside, write-through, eviction
3. **Pub/Sub** - Real-time messaging
4. **Session Store** - Distributed sessions with Spring
5. **Lua Scripting** - Atomic operations
