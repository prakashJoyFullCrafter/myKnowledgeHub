# Kotlin - Expert Level Curriculum

Modern JVM language for backend development.

## Core (Foundation)
1. **Kotlin Syntax** - Basics, control flow, functions, collections, lambdas, scope functions (6 modules)
2. **Data Classes** - Data classes, sealed classes, objects, enums, value classes (6 modules)
3. **Coroutines** - Structured concurrency, flows, channels, exception handling (8 modules)
4. **Extension Functions** - Extensions, properties, DSL building (5 modules)
5. **Null Safety** - Safe calls, Elvis, smart casts, platform types (5 modules)

## Advanced (Expert)
6. **OOP & Generics** - Delegation (`by`), variance (`in`/`out`), reified types, operator overloading (6 modules)
7. **Kotlin-Java Interop** - `@Jvm` annotations, mixed codebases, Spring plugins, coroutine bridging (5 modules)
8. **Kotlin Testing** - MockK, Kotest, coroutine testing, Turbine, Spring integration (6 modules)
9. **Kotlin for Spring Boot** - Idiomatic Spring, coroutine controllers, bean DSL, JPA with Kotlin (8 modules)
