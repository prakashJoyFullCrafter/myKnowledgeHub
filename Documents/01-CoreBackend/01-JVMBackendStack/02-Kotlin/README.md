# Kotlin

Modern JVM language for backend development.

## Topics
1. **Kotlin Syntax** - Basics, control flow, functions, collections, scope functions
2. **Data Classes** - Data classes, sealed classes, objects, enums, value classes
3. **Coroutines** - Structured concurrency, flows, channels
4. **Extension Functions** - Extensions, DSL building
5. **Null Safety** - Safe calls, Elvis, smart casts, platform types
