# JVM Frameworks

Modern JVM backend frameworks for building microservices and APIs.

## Frameworks
1. **Spring Boot** - DI, Data JPA, MVC, REST, Security, Actuator
2. **Quarkus** - Native image, Panache, RESTEasy, Dev Services
3. **Micronaut** - AOT, DI, HTTP Client, GraalVM
4. **Comparison** - Head-to-head: startup, memory, ecosystem
