# Spring Boot - Expert Level Curriculum

The most popular JVM framework for enterprise backend development.

## Core (Foundation)
1. **Spring Core & DI** - IoC, beans, configuration, profiles, auto-configuration
2. **Spring Data JPA** - Entities, repositories, transactions, N+1 problem
3. **Spring MVC** - Controllers, views, form handling, Thymeleaf
4. **Spring REST** - RESTful APIs, DTOs, pagination, OpenAPI, HATEOAS
5. **Spring Security** - Authentication, JWT, OAuth2, authorization
6. **Actuator** - Health checks, metrics, Prometheus, Grafana

## Advanced (Expert)
7. **Spring WebFlux** - Reactive programming, Project Reactor, WebClient, R2DBC
8. **Spring Cloud** - Service Discovery, Gateway, Config Server, Feign, Tracing
9. **Spring Batch** - ETL, chunk processing, partitioning, scheduling
10. **Spring AOP** - Aspects, pointcuts, cross-cutting concerns, proxy internals
11. **Spring Caching** - Caffeine, Redis, cache patterns, multi-level caching
12. **Spring Messaging** - Kafka, RabbitMQ, JMS, application events, outbox pattern
13. **Spring Testing** - MockMvc, Testcontainers, slices, security tests
14. **Scheduling & Async** - `@Scheduled`, `@Async`, ShedLock, thread pools
15. **Profiles & Configuration** - Multi-env, feature flags, secrets, externalized config
16. **Spring Boot Internals** - Auto-configuration, custom starters, lifecycle, production readiness
