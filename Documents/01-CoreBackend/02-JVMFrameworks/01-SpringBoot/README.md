# Spring Boot - Expert Level Curriculum

The most popular JVM framework for enterprise backend development.

## Core (Foundation)
1. **Spring Core & DI** - IoC, beans, configuration, profiles, auto-configuration (5 modules)
2. **Spring Data JPA** - Entities, repositories, transactions, N+1, Flyway/Liquibase, multi-tenancy (9 modules)
3. **Spring MVC** - Controllers, views, form handling, Thymeleaf (5 modules)
4. **Spring REST** - RESTful APIs, DTOs, pagination, OpenAPI, HATEOAS, file upload, WebSocket, gRPC (10 modules)
5. **Spring Security** - Authentication, JWT, OAuth2, authorization, session management (7 modules)
6. **Actuator** - Health checks, metrics, Prometheus, Grafana (5 modules)

## Advanced (Expert)
7. **Spring WebFlux** - Reactive programming, Project Reactor, WebClient, R2DBC (8 modules)
8. **Spring Cloud** - Service Discovery, Gateway, Config Server, Feign, Tracing (8 modules)
9. **Spring Batch** - ETL, chunk processing, partitioning, scheduling (7 modules)
10. **Spring AOP** - Aspects, pointcuts, cross-cutting concerns, proxy internals (5 modules)
11. **Spring Caching** - Caffeine, Redis, cache patterns, multi-level caching (4 modules)
12. **Spring Messaging** - Kafka, RabbitMQ, JMS, application events, outbox pattern (5 modules)
13. **Spring Testing** - MockMvc, Testcontainers, slices, security tests (8 modules)
14. **Scheduling & Async** - `@Scheduled`, `@Async`, ShedLock, thread pools (4 modules)
15. **Profiles & Configuration** - Multi-env, feature flags, secrets, externalized config (5 modules)
16. **Spring Boot Internals** - Auto-configuration, custom starters, lifecycle, production readiness (6 modules)

## Architecture
17. **Spring Modulith** - Modular monolith, module boundaries, events, verification, migration to microservices (6 modules)
