# Micronaut

Compile-time DI framework for cloud-native microservices.

## Build-Time Magic
1. **AOT Compilation** - No runtime reflection, fast startup, performance comparison (4 modules)
2. **Dependency Injection** - Compile-time DI, qualifiers, config, AOP (4 modules)
3. **GraalVM** - Native image building, optimization, deployment (4 modules)

## Building Applications
4. **HTTP Client** - Declarative clients, reactive, filters, testing (4 modules)
5. **HTTP Server** - Controllers, validation, error handling, filters, reactive (5 modules)
6. **Micronaut Data** - Compile-time repositories, JDBC/JPA/R2DBC, transactions (5 modules)
7. **Security** - JWT, OAuth2/OIDC, roles, authorization, testing (5 modules)
