# Quarkus

Supersonic Subatomic Java - cloud-native JVM framework.

## Distinctive Features
1. **Native Image** - GraalVM, AOT compilation, fast startup, limitations (4 modules)
2. **Panache ORM** - Active Record & Repository patterns, queries, transactions (4 modules)
3. **RESTEasy** - Reactive REST, Mutiny, validation, filters (5 modules)
4. **Dev Services** - Auto-provisioned containers, live coding, testing (4 modules)

## Core Framework
5. **CDI (ArC)** - Build-time DI, scopes, qualifiers, interceptors, producers, events (5 modules)
6. **Configuration** - MicroProfile Config, `@ConfigMapping`, profiles, config sources (5 modules)
7. **Security** - JWT, OIDC, Keycloak, roles, permissions, testing (5 modules)
