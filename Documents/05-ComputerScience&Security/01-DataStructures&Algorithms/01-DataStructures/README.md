# Data Structures

Fundamental data structures for efficient problem solving.

1. **Arrays & Strings** - Two pointers, sliding window, prefix sum, KMP, Kadane's
2. **HashMaps** - Hash functions, collisions, two-sum, LRU cache
3. **Heaps** - Min/max heap, priority queue, top-K, merge K sorted
4. **LinkedLists** - Singly/doubly, fast/slow pointer, cycle detection, reverse
5. **Trees & Graphs** - BST, trie, BFS/DFS, Dijkstra, Floyd-Warshall, Union-Find, SCC
6. **Stacks & Queues** - Monotonic stack, deque, sliding window max, priority queue (4 modules)
