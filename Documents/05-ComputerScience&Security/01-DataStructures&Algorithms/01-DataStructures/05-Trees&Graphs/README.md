# Trees & Graphs - Curriculum

## Module 1: Binary Trees
- [ ] Tree terminology: root, leaf, depth, height, level
- [ ] Traversals: inorder, preorder, postorder (DFS)
- [ ] Level-order traversal (BFS)
- [ ] Binary Search Tree (BST): insert, search, delete
- [ ] BST validation
- [ ] Lowest Common Ancestor (LCA)
- [ ] Balanced trees: AVL, Red-Black (concept)

## Module 2: Advanced Trees
- [ ] Trie (prefix tree): insert, search, startsWith
- [ ] Segment tree: range queries
- [ ] Binary Indexed Tree (Fenwick Tree)
- [ ] N-ary trees

## Module 3: Graphs
- [ ] Graph representations: adjacency list, adjacency matrix, edge list
- [ ] BFS and DFS traversal
- [ ] Shortest path: Dijkstra (non-negative weights), Bellman-Ford (negative weights)
- [ ] **Floyd-Warshall**: all-pairs shortest path — O(V^3), detects negative cycles
- [ ] Minimum spanning tree: Kruskal (edge-sorted + Union-Find), Prim (priority queue)
- [ ] Topological sort (DAG): Kahn's algorithm (BFS) or DFS-based
- [ ] Cycle detection (directed: DFS with coloring, undirected: Union-Find or DFS)
- [ ] Union-Find (Disjoint Set Union): path compression + union by rank — near O(1)
- [ ] Connected components (undirected: BFS/DFS/Union-Find)
- [ ] **Strongly Connected Components** (directed): Tarjan's algorithm or Kosaraju's algorithm
- [ ] **Bipartite graph checking**: BFS/DFS with 2-coloring
- [ ] **A* search**: heuristic-guided shortest path (Dijkstra + heuristic)
- [ ] Graph coloring: chromatic number, greedy coloring
- [ ] Euler path/circuit: Hierholzer's algorithm
