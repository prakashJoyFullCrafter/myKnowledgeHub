# Interview Prep - Curriculum

## Topics
- [ ] Problem-solving framework: understand, plan, implement, verify
- [ ] Pattern recognition: sliding window, two pointers, BFS/DFS, DP, greedy
- [ ] Communication: think out loud, clarify requirements, discuss trade-offs
- [ ] Time management: 5 min understand, 10 min plan, 20 min code, 5 min test
- [ ] Edge cases: empty input, single element, duplicates, negative numbers
- [ ] Testing: walk through example, boundary cases
- [ ] System design interview: requirements, estimation, design, deep-dive
- [ ] Behavioral questions: STAR method
- [ ] Top 50 LeetCode patterns for interviews
- [ ] Mock interview practice
