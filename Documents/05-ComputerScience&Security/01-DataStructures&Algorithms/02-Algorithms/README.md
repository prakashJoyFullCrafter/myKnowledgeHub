# Algorithms

Core algorithmic techniques and patterns.

1. **Sorting** - Bubble, Selection, Insertion, Merge, Quick, Heap, Counting, Radix
2. **BFS & DFS** - Trees, graphs, grids, multi-source BFS
3. **Dynamic Programming** - Memoization, tabulation, knapsack, string DP, bitmask DP
4. **Greedy** - Activity selection, interval scheduling, Huffman, correctness proofs
5. **Backtracking** - Permutations, N-Queens, Sudoku, constraint satisfaction
6. **Binary Search** - Classic, search on answer, rotated array, median of two arrays (4 modules)
7. **Bit Manipulation** - Bitwise ops, XOR tricks, single number, bitmask DP (4 modules)
