# Sorting Algorithms - Curriculum

## Topics
- [ ] Bubble Sort: O(n^2), simple, stable
- [ ] Selection Sort: O(n^2), in-place
- [ ] Insertion Sort: O(n^2), good for nearly sorted
- [ ] Merge Sort: O(n log n), stable, divide-and-conquer
- [ ] Quick Sort: O(n log n) avg, in-place, pivot selection
- [ ] Heap Sort: O(n log n), in-place
- [ ] Counting Sort: O(n+k), non-comparison
- [ ] Radix Sort: O(d*(n+k)), digit-based
- [ ] Comparison table: time, space, stability
- [ ] Java: `Arrays.sort()` (dual-pivot quicksort), `Collections.sort()` (timsort)
