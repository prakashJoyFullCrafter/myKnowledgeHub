# Reconnaissance & Enumeration - Curriculum

- [ ] Passive recon: OSINT, WHOIS, DNS records, Shodan, Google dorking
- [ ] Active recon: port scanning, service detection, OS fingerprinting

- [ ] Directory/file brute-forcing: `gobuster`, `dirb`, `ffuf`
- [ ] Web technology fingerprinting: `whatweb`, `wappalyzer`
- [ ] SMB/LDAP/SNMP enumeration
- [ ] Banner grabbing and version detection
- [ ] Vulnerability scanning: `nessus`, `openvas`
- [ ] Mapping attack surface
