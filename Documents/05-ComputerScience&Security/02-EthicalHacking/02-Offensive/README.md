# Offensive Security

Penetration testing techniques and methodologies.

### Foundations (for backend devs)
1. **Recon & Enumeration** - OSINT, port scanning, subdomain enum, directory brute-force
2. **SQL Injection** - In-band, blind, second-order, sqlmap, prevention
3. **XSS & CSRF** - Reflected/stored/DOM XSS, CSRF tokens, SameSite cookies
4. **Exploitation** - LFI/RFI, command injection, privilege escalation, post-exploitation
5. **API Security (OWASP API Top 10)** - BOLA, broken auth, mass assignment, SSRF, BFLA (10 modules)

### Advanced (for pentest/OSCP/red team path)
6. **Advanced Web Exploitation** - Deserialization, SSTI, XXE, JWT, OAuth, HTTP smuggling, prototype pollution, SSRF deep dive (10 modules)
7. **Privilege Escalation** - Linux + Windows enumeration, sudo/SUID abuse, GTFOBins/LOLBAS, kernel exploits, token impersonation (6 modules)
8. **Active Directory Attacks** - Kerberoasting, ASREPRoast, NTLM relay, BloodHound, DCSync, ACL abuse, ADCS (7 modules)
9. **Network Protocol Attacks** - ARP/DNS spoofing, LLMNR/NBT-NS poisoning (Responder), mitm6, IPv6 attacks (6 modules)
10. **Red Team Basics & MITRE ATT&CK** - 14 tactics, initial access, persistence, C2, lateral movement, reporting (8 modules)
