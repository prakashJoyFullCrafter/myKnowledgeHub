# Ethical Hacking

Security testing, penetration testing, and CTF preparation.

## Sections
1. **Foundations** (5 topics) - Linux, Networking, OWASP Top 10, Cryptography, Cloud & Container Security
2. **Offensive** (10 topics) - Recon, SQLi, XSS/CSRF, Exploitation, API Security, Advanced Web, Privilege Escalation, AD Attacks, Network Protocol Attacks, Red Team Basics
3. **Tooling** (5 topics) - Kali Linux, Nmap, Burp Suite, Metasploit, CTF

## Learning Paths

**Backend dev (defensive awareness)**: Topics 1-5 of Foundations + 1-5 of Offensive = 10 topics

**Bug Bounty Hunter**: Add topic 6 (Advanced Web Exploitation) — focus on PortSwigger Web Security Academy

**OSCP Preparation**: All 20 topics + heavy HackTheBox/TryHackMe practice

**Red Team / CRTP / CRTO**: All 20 topics + HackTheBox Pro Labs (RastaLabs, Offshore, Dante)
