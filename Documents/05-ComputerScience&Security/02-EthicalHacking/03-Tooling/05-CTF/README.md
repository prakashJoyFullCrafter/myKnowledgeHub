# CTF (Capture The Flag) - Curriculum

## Topics
- [ ] CTF types: Jeopardy, Attack-Defense, King of the Hill
- [ ] Categories: web, crypto, reverse engineering, forensics, pwn, misc
- [ ] Platforms: HackTheBox, TryHackMe, PicoCTF, CTFtime
- [ ] Web challenges: SQLi, XSS, SSRF, deserialization
- [ ] Crypto challenges: classical ciphers, RSA, AES, hashing
- [ ] Forensics: file carving, steganography, memory analysis
- [ ] Reverse engineering: `ghidra`, `gdb`, `radare2`, `IDA`
- [ ] Binary exploitation: buffer overflow, ROP chains
- [ ] Building a CTF toolkit
- [ ] Writeup practice: documenting solutions
