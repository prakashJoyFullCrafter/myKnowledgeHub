# Nmap - Curriculum

## Topics
- [ ] Host discovery: `-sn`, `-Pn`, `-PS`, `-PA`
- [ ] Port scanning: `-sS` (SYN), `-sT` (TCP), `-sU` (UDP)
- [ ] Service detection: `-sV`
- [ ] OS detection: `-O`
- [ ] Script scanning: `-sC`, `--script`
- [ ] NSE scripts: `vuln`, `auth`, `discovery`
- [ ] Output formats: `-oN`, `-oX`, `-oG`
- [ ] Timing: `-T0` to `-T5`
- [ ] Firewall evasion techniques
- [ ] Common scan recipes for pentesting
