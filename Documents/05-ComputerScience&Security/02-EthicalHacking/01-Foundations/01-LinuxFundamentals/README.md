# Linux Fundamentals - Curriculum

## Topics
- [ ] File system hierarchy: `/etc`, `/var`, `/home`, `/usr`, `/bin`, `/tmp`
- [ ] File permissions: `chmod`, `chown`, `chgrp`, rwx, SUID/SGID
- [ ] User management: `useradd`, `usermod`, `passwd`, `/etc/passwd`, `/etc/shadow`
- [ ] Process management: `ps`, `top`, `htop`, `kill`, `systemctl`
- [ ] Networking: `ifconfig`, `ip`, `netstat`, `ss`, `curl`, `wget`
- [ ] Package management: `apt`, `yum`, `dnf`
- [ ] Shell scripting basics: variables, loops, conditionals, functions
- [ ] Text processing: `grep`, `sed`, `awk`, `cut`, `sort`, `uniq`
- [ ] SSH: key generation, config, tunneling, SCP
- [ ] Cron jobs and task scheduling
