# Security Foundations

Prerequisites for ethical hacking and security testing.

1. **Linux Fundamentals** - File system, permissions, processes, networking, shell scripting
2. **Networking Basics** - OSI/TCP/IP, TCP handshake, DNS, HTTPS, Wireshark
3. **OWASP Top 10** - All 10 web app vulnerabilities + mitigations
4. **Cryptography** - Hashing, AES, RSA/ECC, TLS, JWT, common mistakes (6 modules)
5. **Cloud & Container Security** - Docker, K8s, IAM, SSRF/metadata, supply chain (6 modules)
