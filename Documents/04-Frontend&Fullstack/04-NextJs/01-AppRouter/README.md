# Next.js App Router - Curriculum

## Topics
- [ ] App Router vs Pages Router (Next.js 13+)
- [ ] File-based routing: `app/page.tsx`, `app/about/page.tsx`
- [ ] Layouts: `layout.tsx` - shared UI across routes
- [ ] Templates: `template.tsx` - re-mount on navigation
- [ ] Loading UI: `loading.tsx` with Suspense
- [ ] Error handling: `error.tsx` with Error Boundary
- [ ] Not found: `not-found.tsx`
- [ ] Route groups: `(group)/page.tsx` - organize without affecting URL
- [ ] Dynamic routes: `[slug]/page.tsx`, `[...slug]`, `[[...slug]]`
- [ ] Parallel routes: `@modal/page.tsx`
- [ ] Intercepting routes: `(.)photo/[id]`
