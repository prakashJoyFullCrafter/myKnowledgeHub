# Frontend & Fullstack

Modern frontend development with JavaScript/TypeScript and React ecosystem.

## Sections

### Foundations
- **01. ES6+** - Modern JavaScript fundamentals
- **02. TypeScript** - Types, generics, decorators, utility types

### React Ecosystem
- **03. React.js** - Hooks, Context, state management, React Query
- **04. Next.js** - App Router, SSR/SSG/ISR, Server Components, API routes

### Production-Grade & Architect Topics
- **05. Performance & Web Vitals** - Core Web Vitals, code splitting, memoization, virtualization & streaming
- **06. Testing** - Unit (Jest/Vitest), Component (RTL), E2E (Playwright/Cypress), Mocking (MSW)
- **07. Build Tooling** - Vite, Webpack, Turbopack/esbuild, Monorepos (pnpm/Turbo/Nx)
- **08. Auth** - NextAuth/Auth.js, JWT & Sessions, OAuth/OIDC, RBAC & Authorization
- **09. Frontend Architecture** - Micro-Frontends, Module Federation, BFF pattern, Design Systems
