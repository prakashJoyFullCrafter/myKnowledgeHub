# TypeScript Generics - Curriculum

## Topics
- [ ] Generic functions: `function identity<T>(arg: T): T`
- [ ] Generic interfaces and classes
- [ ] Generic constraints: `<T extends HasLength>`
- [ ] `keyof` operator: `<T, K extends keyof T>`
- [ ] Generic utility types: `Partial<T>`, `Required<T>`, `Readonly<T>`
- [ ] Mapped types: `{ [K in keyof T]: boolean }`
- [ ] Conditional types: `T extends U ? X : Y`
- [ ] `infer` keyword in conditional types
- [ ] Generic default types: `<T = string>`
- [ ] Generic constraints with multiple types
- [ ] Real-world: generic API response wrapper, generic form handler
