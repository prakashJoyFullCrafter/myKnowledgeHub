# ES6 Modules - Curriculum

## Topics
- [ ] Named exports: `export const foo = ...`
- [ ] Default exports: `export default class ...`
- [ ] Named imports: `import { foo } from './module'`
- [ ] Default imports: `import MyClass from './module'`
- [ ] Renaming imports: `import { foo as bar }`
- [ ] `import * as module from './module'`
- [ ] Re-exporting: `export { foo } from './other'`
- [ ] Dynamic imports: `import('./module')` (lazy loading)
- [ ] CommonJS (`require`) vs ES Modules (`import`)
- [ ] Module resolution in Node.js and bundlers
