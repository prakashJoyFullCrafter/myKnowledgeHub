# Array Methods - Curriculum

## Topics
- [ ] `map()` - transform each element
- [ ] `filter()` - select elements matching condition
- [ ] `reduce()` - accumulate to single value
- [ ] `forEach()` - side effects (no return value)
- [ ] `find()` and `findIndex()` - first match
- [ ] `some()` and `every()` - boolean tests
- [ ] `flat()` and `flatMap()` - flatten nested arrays
- [ ] `includes()` - check membership
- [ ] `Array.from()` - create array from iterable
- [ ] `Array.of()` - create array from arguments
- [ ] Method chaining: `arr.filter().map().reduce()`
- [ ] Performance: when to use `for` loop instead
