# Kubernetes - Curriculum

## Module 1: Core Concepts
- [ ] Architecture: Control Plane (API Server, etcd, Scheduler, Controller Manager) + Worker Nodes (kubelet, kube-proxy)
- [ ] Pod: smallest deployable unit
- [ ] Deployment: declarative updates, rolling updates, rollbacks
- [ ] Service: ClusterIP, NodePort, LoadBalancer, ExternalName
- [ ] Namespace: logical isolation

## Module 2: Configuration & Storage
- [ ] ConfigMap and Secret
- [ ] PersistentVolume (PV) and PersistentVolumeClaim (PVC)
- [ ] StorageClass: dynamic provisioning
- [ ] Resource requests and limits (CPU, memory)

## Module 3: Networking & Ingress
- [ ] Ingress Controller and Ingress resources
- [ ] Network Policies for pod-level firewall
- [ ] Service mesh integration (Istio, Linkerd)
- [ ] DNS in Kubernetes

## Module 4: Advanced
- [ ] StatefulSet: stateful workloads (databases)
- [ ] DaemonSet: one pod per node (logging, monitoring agents)
- [ ] Jobs and CronJobs
- [ ] Horizontal Pod Autoscaler (HPA)
- [ ] RBAC: Role, ClusterRole, RoleBinding
- [ ] Probes: liveness, readiness, startup
- [ ] `kubectl` essential commands
