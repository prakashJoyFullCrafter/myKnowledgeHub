# GitHub Actions - Curriculum

## Topics
- [ ] Workflow YAML syntax: `on`, `jobs`, `steps`, `runs-on`
- [ ] Triggers: `push`, `pull_request`, `schedule`, `workflow_dispatch`
- [ ] Runners: GitHub-hosted, self-hosted
- [ ] Actions marketplace and reusable actions
- [ ] Secrets and environment variables
- [ ] Matrix builds (multiple OS, Java versions)
- [ ] Caching dependencies (`actions/cache`)
- [ ] Artifacts: upload/download between jobs
- [ ] Reusable workflows and composite actions
- [ ] CI pipeline: build, test, lint, security scan
- [ ] CD pipeline: deploy to staging/production
- [ ] Environment protection rules and approvals
