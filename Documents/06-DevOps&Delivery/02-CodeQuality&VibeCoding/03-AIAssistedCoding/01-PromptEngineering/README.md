# Prompt Engineering - Curriculum

## Topics
- [ ] Clear and specific instructions
- [ ] Providing context: codebase, constraints, requirements
- [ ] Few-shot examples for consistent output
- [ ] Chain-of-thought prompting for complex problems
- [ ] System prompts for persona and behavior
- [ ] Iterative refinement of prompts
- [ ] Code generation: specifying language, framework, patterns
- [ ] Code review prompts: asking for specific feedback
- [ ] Debugging prompts: providing error context
- [ ] Limitations: hallucination, outdated knowledge, context window
